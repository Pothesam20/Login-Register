# Login-Register Full Stack App

React + Spring Boot + H2 Database

## Quick Start

### Backend
```bash
cd loginregister-backend
start.bat
```
Wait for: `Started UserProfileApplication`

### Frontend (New Terminal)
```bash
cd login-register
start.bat
```
Opens at: http://localhost:3000

## Tech Stack
- Frontend: React 19
- Backend: Spring Boot 3.2
- Database: H2 (in-memory)
- Port: Backend 8080, Frontend 3000
